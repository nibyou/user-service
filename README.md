# Nibyou Microservice

## Installation

```bash
$ npm install
```

## Configuration

Check out the `example.env` file in the root directory.

Fill out the variables and rename the file to `.env`.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
